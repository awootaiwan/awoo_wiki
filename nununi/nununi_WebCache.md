# Web Cache

![](./images/web_cache.jpeg)


## awoo 建議您的 Cache 機制

**(步驟 1)** 當頁面被請求時，程式先檢查有無快取內容

 ## 若無快取內容

 * **(步驟 2)** 呼叫 awoo API 取得最新資料
 * 將資料存於快取存放處，並賦予快取資料時間戳記
 * **(步驟 4)** 程式回應頁面內容

 ## 若有快取內容

  若有快取內容，將會建議『檢查快取資料是否過期』

  ###  (步驟 3)若未過期

  **(步驟 4)** 程式由快取存放處取用資料回應頁面內容

  ###  (步驟 3)若過期

   * **(步驟 2)** 呼叫 awoo API 取得最新資料
   * 將資料存於快取存放處
   * 更新快取資料時間戳記
   * **(步驟 4)** 程式回應頁面內容
